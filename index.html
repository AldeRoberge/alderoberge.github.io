Technologies maîtrisés

Unity

J'ai développé mes conaissances sur Unity et ses différentes capacités.

HTML et CSS

J'ai fait mon portfolio en HTML et CSS.

MongoDB

J'utilise un wrapper asynchrone de MongoDB pour la base de données.

LiteNetLib

Le jeu est multijoueur. Basé sur un système de transport de bas niveau (en UDP), j'ai construit un système multijoueur.

Discord API

Le jeu est fortement intégré avec un robot Discord. Les joueurs peuvent interagir avec ce robot et voir leurs statistiques et recevoir des notifications en lien avec le jeu directement par le biais de Discord.

Stripe API

J’ai aussi mis en place un système de paiement qui permet au joueur d’acheter des éléments avec de l’argent (Stripe API) sur le site web et de retrouver ses achats dans le jeu.

Internationalisation

Le jeu est disponible en anglais, français et espagnol. J’ai construit un système d'internationalisation qui permet d’ajouter de nouvelle langues automatiquement, en utilisant l’API de Google Translate.
